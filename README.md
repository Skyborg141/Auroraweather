# 🌤️ AuroraWeather

A beautiful, browser-based weather app with real-time forecasts, animated visuals, and city search.

🔗 **[Live Demo](https://skyborg141.github.io/Auroraweather/)**

---

## Features

- 📍 Auto-detect location via GPS
- 🔍 City search with autocomplete
- 🌡️ Current weather — temperature, feels like, humidity, wind, pressure, visibility, UV index
- ⏱️ Hourly forecast
- 📅 7-day forecast
- 🌅 Sunrise & sunset arc tracker
- ✨ Animated weather backgrounds

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- Vite (build tool)
- Deployed via GitHub Pages

## Getting Started

```bash
git clone https://github.com/Skyborg141/Auroraweather.git
cd Auroraweather
npm install
npm run dev
```
